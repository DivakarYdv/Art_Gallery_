# Art_Gallery_

Online Art Gallery
A full-stack web application for showcasing and selling original artwork. Built from scratch with no frontend frameworks — just vanilla HTML, CSS, and JavaScript on the frontend, and Node.js, Express, and MySQL on the backend.

Layer                        Technology
Frontend                     HTML, CSS (Grid + Flexbox), Vanilla JavaScript
Backend                      Node.js, Express
Database                     MySQL
Styling                      Custom CSS with design tokens, no frameworks
